## 活动规则

本活动为 **算法对战**，请使用你自己的程序接入，我们提供0门槛的 SDK ，只要你懂 C++ 语法，就能参与！

### 参与步骤

1. 打开 [活动主页](https://codingsnake.seveoi.icu)。
2. 按 认证说明 获取游戏凭证 uid 和 paste。
4. 使用 SDK 或 HTTP API 编写 Bot 并加入对局。
5. 在前端实时观战，查看排行榜
